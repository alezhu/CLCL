CLCL clipboard manager
======================
CLCL is a software that records clipboard history and offers means to store text templates and various tools to enhance the usage of the clipboard.

This installation package has been compiled from https://github.com/wilfz/CLCL.

This repository is a fork of https://github.com/nakkag/CLCL and includes fixes and new features.

The installation package includes the CLCL main program and some useful plugins.

All above components are compiled with the latest platform toolset v143.
